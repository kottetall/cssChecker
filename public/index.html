Check my CSS

This page is ment to be a testing site to check how backwards compatible your CSS is.

You'll be able to see the oldest version that your CSS supports as well as which properties is 'holding you back'.

This check picks out all the properties in your CSS and crosschecks them against the properties that's available through MDNs Github repo. Then it searches which version(if any) of a specific browser, was the first to start supporting the property.
